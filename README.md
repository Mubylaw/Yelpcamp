## General info
This is a campground app where users can upload different campgrounds. It is built with NodeJs, express and mongoDb.
	
## Technologies
Project is created with:
* express version: 4.17.1
* mongoose version: 5.5.14
* passport version: 0.4.0

## Features
List of features ready and TODOs for future development
* Users can sign up and sign in to the app
* Users can create, view and delete campgrounds
* Users can comment on different campgrounds and delete comments they have authorisation to delete.

To-do list:
* Add an authomatical mail system possibly mailchimp
* Add a payment system

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ node app.js
```
